Detailed Description

List of possible systematic uncertainties for the hlfv study.

Essentially a copy of what we used for susy::wh, with a few additional helper functions to handle names and to convert between hlfv::Systematic <-> SusyNtSys

Note to self: enclose enum in a struct to avoid collisions; see explanation on stackoveflow.
